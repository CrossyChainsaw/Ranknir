<h1 align="center"><b>WHAT IS RANKNIR?</b></h1>
<p align="center"><b>Ranknir is a Discord bot that creates elo leaderboards for Brawlhalla clans and organisations, supporting 1v1, 2v2, and rotational game modes.</b></p>

<h2 align="center"><b>HOW TO USE RANKNIR</b></h2>
<p align="center"><b>USE THE SERVICE</b> - Ranknir runs 24/7 and is fully managed for you in exchange for a small yearly fee.</p>
<p align="center"><b>SELF-HOST FOR FREE</b> - The code is fully open source. Grab your own Brawlhalla API key, run it on your own server, and it costs you nothing. Can't figure it out? Reach out on Discord: <b>crossychainsaw</b></p>
